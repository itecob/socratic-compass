# Session Handoffs

This directory holds structured end-of-session notes, written by `compass:session-handoff` at the end of any session that modified code or made structural decisions.

**Format:** `YYYY-MM-DD-HHMM.md`

**Contents per handoff:** what changed structurally, what is now true that wasn't, what is half-finished, what the next session needs to know, open questions.

See the Compass plugin's `session-handoff` skill for the structure. Keep this README as a directory marker or remove it once you have real handoffs here.
