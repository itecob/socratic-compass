# Premise-Checks

This directory holds `compass:premise-check` reports. One file per planning cycle, parallel to `interviews/`.

**Format:** `YYYY-MM-DD-HHMM.md`

**Contents per report:** the five outputs of premise-check (two alternative framings, load-bearing assumptions, pivotal question, validation against the architecture journal, categorization), plus the human's response (confirmation, corrections, overrides).

Each premise-check references the originating interview transcript at `.architecture/interviews/`. The pair forms a complete planning record.

See the Compass plugin's `premise-check` skill for the discipline that produces these. Keep this README as a directory marker or remove it once you have real premise-checks here.
