# Interviews

This directory holds `compass:socratic-interview` transcripts. One file per planning session.

**Format:** `YYYY-MM-DD-HHMM.md`

**Contents per transcript:** problem statement (in the human's words), constraints committed to, decisions made, open questions deferred, gaps the agent attempted (with the human's response to each), and the agent's final assessment.

See the Compass plugin's `socratic-interview` skill for the discipline that produces these. The first transcript will replace this README in spirit — keep this file as a directory marker or remove it once you have real interviews here.
